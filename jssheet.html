JS Cheat Sheet


Command Description
console.log() Outputs a message to the browser console.
document.getElementById() Returns the element with the specified ID.
document.querySelector() Returns the first element that matches a specified CSS selector.
document.createElement() Creates a new HTML element.
element.innerHTML Gets or sets the HTML content of an element.
element.classList.add() Adds one or more classes to an element.
element.classList.remove() Removes one or more classes from an element.
element.addEventListener() Attaches an event handler to the specified element.
window.onload() Specifies a function to run after the document has been loaded.
setTimeout() Calls a function or evaluates an expression after a specified number of milliseconds.
setInterval() Repeats a function or executes an expression at specified intervals.
JSON.stringify() Converts a JavaScript object into a JSON string.
JSON.parse() Parses a JSON string, constructing the JavaScript value or object described by the string.
typeof Returns the type of a variable or expression.
var, let, const Keywords used to declare variables.
function Declares a function.
if...else Conditionally executes code based on a condition.
for, while Loops that execute code repeatedly.
try...catch Handles errors in code blocks.
return Exits a function and specifies a value to be returned to the caller.
NaN Represents "Not-a-Number".
null Represents the intentional absence of any object value.
undefined Represents an undefined value.
this Refers to the current object in a method or constructor.
new Creates an instance of a user-defined object type or built-in object type.
Object.keys() Returns an array of a given object's own enumerable property names.
new Date() Creates a new Date object representing the current date and time.
new Date(year, month, day, hours, minutes, seconds, milliseconds) Creates a new Date object with the specified date and time components.
getTime() Returns the numeric value corresponding to the time for the specified date according to universal time.
clearInterval() Clears a timer set with the setInterval() method.
Math.floor() Returns the largest integer less than or equal to a given number.
Math.random() Returns a random number between 0 (inclusive) and 1 (exclusive).
