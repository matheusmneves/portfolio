HTML Cheat Sheet


Element Description
<html> Root element of an HTML document, containing all other elements.
<body> Contains the content of the HTML document visible to the user.
<head> Contains metadata about the HTML document, such as title, character set, etc.
<link> Links an external resource to the HTML document, such as a CSS stylesheet or a favicon.
<style> Defines internal CSS styles within an HTML document.
<meta> Specifies metadata about an HTML document, such as character encoding, viewport settings, etc.
<nav> Navigation element containing navigation links.
<section> Defines a section on the page, such as profile, about, experience, projects, etc.
<div> Used to group elements and apply styles.
<img> Used to display images on the page.
<p> Defines a paragraph of text.
<h1> to <h6> Defines headings of different levels of importance.
<ul> and <li> Used to create unordered lists.
<ol> and <li> Used to create ordered lists.
<button> Creates a clickable button.
<script> Defines a script, usually used to include JavaScript scripts.
<a> Defines a hyperlink, used to link to another page or resource.
<span> Inline container used to apply styles to a specific portion of text.
<table> Defines a table, used to display data in rows and columns.
<tr> Defines a row in a table.
<th> Defines a header cell in a table.
<td> Defines a data cell in a table.
<thead> Defines the header of a table.
<tbody> Defines the body of a table.
<tfoot> Defines the footer of a table.
<iframe> Embeds another document within the current HTML document.
<span> Inline container used to apply styles to a specific portion of text.
<strong> Emphasizes text, typically displayed in bold.
<em> Emphasizes text, typically displayed in italic.
<br> Inserts a single line break.
<hr> Creates a horizontal rule, used to separate content.
