CSS Cheat Sheet


Selector / Property Description
* Selects all elements
body Selects the body element
html Represents the root of the document
p Defines a paragraph
a Defines a hyperlink
.class Selects elements with a specific class
#id Selects an element with a specific id
hover Applies styles when hovering over an element
transition Adds smooth transitions to property changes
@import Imports external style sheets
font-family Sets the font family for text
color Sets the color of text
background-color Sets the background color of an element
width Sets the width of an element
height Sets the height of an element
margin Sets the margin around an element
padding Sets the padding inside an element
border Sets the border of an element
display Sets how an element is displayed
flex Specifies the ability of a flex item to alter its dimensions to fill the available space
justify-content Defines the alignment along the main axis
align-items Defines the alignment along the cross axis
gap Specifies the gap between flex items
text-align Specifies the horizontal alignment of text
border-radius Defines the radius of the element's corners
text-decoration Adds decoration to text, such as underline or strikethrough
cursor Specifies the type of cursor to be displayed when hovering over an element
